Usage: [-v|--verbose] [-h|--help] { *Scale | *Rotate }  <file> ...
*Scale: scale [--debug] {-p|--percent <percent>}
*Rotate: rotate [--debug] [--ccw]

Common Options

  --ccw            Rotate counterclockwise

  --debug          Debug output

  -h, --help       Print help

  -p, --percent    Scale percentage

  -v, --verbose    Increase verbosity level

  <file>           Files to process
